### Blender - Easily toggle Brush/Eraser in Texture Paint Mode

This addon adds some useful keybinds to toggle between brush and eraser in texture paint mode, and some other keybinds.

- B - Toggle Brush (Mix)
- E - Toggle Brush (Erase Alpha)
- W - Toggle Soften 
- D - Toggle Smear
- C - Toggle Clone
- G - Toggle Fill
- Space - Context Menu
